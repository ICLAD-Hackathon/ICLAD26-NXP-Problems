Secure Peripheral Subsystem SoC

EASYiverilogAPB3 · AHB-LiteArchitecture reference — diagrams + minimal spec

Problem Statement

  1. Read the diagrams below to understand the SoC architecture and each IP block's behavior.
  2. Infer the YAML spec for each IP block from the diagrams (e.g., FIFO depth, channel count, timeout cycles).
  3. Use the RTL generation library: python3 rtl_gen_lib/rtl_gen_main.py --spec <yaml> --outdir your_rtl/
  4. Write secure_periph_soc.v to stitch all IPs together per the SoC Top-Level diagram.
  5. Compile and simulate: iverilog -g2005 -o sim your_rtl/*.v tb/tb_top_skeleton.v && vvp sim
01

SoC Top-Level Interconnect

graph
The CPU connects via AHB-Lite through a bridge to an APB fabric with 5 peripheral slaves. Interrupt lines flow back through the IRQ aggregator. Reset propagates through a synchronizer. Use this diagram to wire up secure_periph_soc.v.
02

Reset Synchronizer

3-stage sync
Generates sys_rst_n: asserts asynchronously on POR or WDT reset, deasserts synchronously after N flip-flop stages. Infer stages from the diagram.
03

AHB→APB Bridge FSM

state machine
Translates an AHB NONSEQ transaction into an APB SETUP+ENABLE cycle. hready=0 stalls the CPU while APB is busy. PSLVERR generates a 2-cycle AHB ERROR response.
04

APB Fabric Address Decode

flowchart
Routes APB master to one of 5 slaves based on paddr[31:12]. Slave 3 (Watchdog) requires pprot[0]=1 (privileged). Any non-responding slave times out after 16 cycles → PSLVERR. Infer timeout_cyc.
05

APB UART — apb_uart

0x0000_0000
Full-duplex UART with separate TX and RX FIFOs. Infer fifo_depth from the FIFO labels. baud_rate = clk / (16 × (baud_div+1)); infer default_div from CTRL register description.
Register Map
OffsetNameAccessDescription
0x000TXDATAWOPush byte to TX FIFO
0x004RXDATAROPop byte from RX FIFO
0x008STATUSRO[0]=tx_full [1]=tx_empty [2]=rx_full [3]=rx_empty [4]=frame_err [5]=parity_err [6]=overrun [7]=cts_n
0x00CCTRLRW[0]=tx_en [1]=rx_en [2]=parity_en [3]=parity_odd [4]=stop2 [23:8]=baud_div
0x010IRQENRWInterrupt enable mask (mirrors STATUS bits)
0x014IRQSTATW1CInterrupt status — write 1 to clear
06

APB GPIO — apb_gpio

0x0000_1000
32 bidirectional pins with 3-stage input debounce and per-pin edge/level IRQ. Infer gpio_width from the pin bus label.
Register Map
OffsetNameAccessDescription
0x000DATA_INRODebounced input values
0x004DATA_OUTRWOutput data
0x008DIRRWDirection: 1=output, 0=input per pin
0x00CALT_LORW2-bit alt func, pins [15:0]
0x010ALT_HIRW2-bit alt func, pins [31:16]
0x014IRQ_ENRWEnable per pin
0x018IRQ_EDGERW0=level, 1=edge
0x01CIRQ_POLRW0=low/fall, 1=high/rise
0x020IRQ_STATW1CPending status — write 1 to clear

APB Timer — apb_timer

0x0000_2000
Two independent 32-bit down-counters with 8-bit prescaler and PWM output. Infer channels from the Channel 0 / Channel 1 subgraphs.

OffsetNameAccessDescription
0x000/0x020LOADRW32-bit reload value CH0/CH1
0x004/0x024VALUEROCurrent counter value
0x008/0x028CTRLRW[0]=en [1]=periodic [2]=irq_en [3]=pwm_en [11:4]=prescale
0x00C/0x02CCOMPARERWPWM compare threshold
0x010/0x030IRQSTATW1CTerminal count flag
08

APB Watchdog FSM — apb_watchdog

0x0000_3000 ⚠ PRIVILEGED
Privileged access only. Write 0xABCD_1234 to UNLOCK before configuring. Kick the watchdog with 0xFEED_C0DE to REFRESH. Two-stage timeout: Stage-1 fires IRQ, Stage-2 asserts system reset. Window mode: early kick = violation.
Register Map
OffsetNameAccessDescription
0x000LOAD1RW*Stage-1 timeout (requires unlock)
0x004LOAD2RW*Stage-2 timeout (requires unlock)
0x008VALUEROCurrent counter
0x00CCTRLRW*[0]=en [1]=window_en [2]=rst_en [3]=irq_en
0x010STATUSRO[0]=irq_pending [1]=in_window [2]=unlocked
0x014UNLOCKWOWrite 0xABCD_1234 (~15 cycle window)
0x018REFRESHWOWrite 0xFEED_C0DE to kick
0x01CIRQSTATW1C[0]=stage1_fired [1]=window_violation
09

IRQ Aggregator — irq_aggregator

0x0000_4000
Collects 8 interrupt sources with configurable edge/level and polarity. Priority encoder selects the highest pending source (src7=highest). Outputs cpu_irq and cpu_irq_id[2:0] vector to the CPU.
Register Map
OffsetNameAccessDescription
0x000IRQ_RAWRORaw unmasked inputs
0x004IRQ_PENDROPending masked+latched
0x008IRQ_ENRWEnable mask
0x00CIRQ_EDGERW0=level, 1=edge per src
0x010IRQ_POLRW0=active-low/fall, 1=active-high/rise
0x014IRQ_CLRWOWrite 1 to clear edge latch
0x018IRQ_VECRO[2:0] vector ID of highest pending
0x01CIRQ_SOFTRWSW interrupt trigger

Address Map — 20KB Peripheral Space

memory map
Each slave occupies a 4KB window. Access Watchdog (S3) only with hprot[0]=1. All other slaves accept any privilege level. Accesses to unmapped addresses return PSLVERR immediately.
Register Map
OffsetNameAccessDescription
0x0000_0000–0x0FFFapb_uart S0RWAny privilege · 6 registers
0x0000_1000–0x1FFFapb_gpio S1RWAny privilege · 9 registers
0x0000_2000–0x2FFFapb_timer S2RWAny privilege · 5 regs × 2 channels
0x0000_3000–0x3FFFapb_watchdog S3RW*PRIVILEGED ONLY · 8 registers · unlock required
0x0000_4000–0x4FFFirq_aggregator S4RWAny privilege · 8 registers
0x0000_5000+— unmapped ——Returns PSLVERR immediately
